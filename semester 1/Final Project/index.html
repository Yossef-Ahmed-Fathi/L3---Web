HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It provides the structure of a webpage, allowing developers to define elements such as headings, paragraphs, links, images, and more.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, and fonts of web pages, allowing developers to create visually appealing designs.

Bootstrap

Bootstrap is a free and open-source CSS framework directed and designed for responsive, mobile-first web development. It contains CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

JavaScript

JavaScript is a programming language that enables interactive web pages. It is an essential technology for web development, allowing developers to create dynamic content and handle user interactions.
